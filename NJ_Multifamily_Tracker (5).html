Make This a Permanent Link

Host this file for free in ~5 minutes. Your API key travels with the file.

1

Get Your Anthropic API Key

Go to console.anthropic.com/keys, sign up free, and create a new key. It looks like sk-ant-api03-.... You get free credits to start.

2

Download This File

Click ↓ Export CSV to save your listings, then download this HTML file. Rename it something like tracker.html.

3

Option A — Netlify Drop (Easiest, 60 seconds)

Go to app.netlify.com/drop, drag and drop your tracker.html file. You instantly get a live URL like amazing-name-123.netlify.app. Free forever. You can claim a custom name too.

4

Option B — GitHub Pages (Free, more control)

Create a free GitHub account → New repo → Upload tracker.html → Rename it index.html → Go to Settings → Pages → Deploy from main branch. Your URL: yourusername.github.io/reponame

5

Enter Your API Key on the Live Page

Once hosted, open your live URL, paste your API key in the 🔑 key field at the top, and click Save. It's stored in your browser locally — never sent anywhere except directly to Anthropic when you hit AI Refresh. Works on any device where you enter the key.

6

Bookmark It

Bookmark your live URL. Share it with anyone you want (they'll need their own API key to use AI Refresh, but can view and add listings freely).
